# Keyboard Shortcuts

## Global Navigation

| Shortcut | Action |
| --- | --- |
| `Ctrl + 1` | Open Servers tab |
| `Ctrl + 2` | Open Status tab |
| `Ctrl + 3` | Open Docker tab |
| `Ctrl + 4` | Open Terminal tab |
| `Ctrl + 5` | Open Snippets tab |
| `Ctrl + 6` | Open Alerts tab |
| `Ctrl + 7` | Open Settings tab |
| `Tab` | Switch to the next main tab |

The same shortcut list is available in the app under `Settings > Shortcuts`.

## Terminal

| Shortcut | Action |
| --- | --- |
| `Ctrl + S` | Open Snippets/Commands palette |
| `Esc` | Close Snippets/Commands palette |
| `Ctrl + H` | Open command history |
| `Esc` | Close command history |
| `Ctrl + T` | Open a new terminal tab for the selected server |
| `Ctrl + Shift + T` | Close the active terminal tab |
| `Ctrl + Shift + C` | Copy selected terminal text |
| `Ctrl + Shift + V` | Paste clipboard into terminal |

## Status And Docker

| Shortcut | Action |
| --- | --- |
| `Ctrl + R` | Refresh the current Status or Docker view |
| `Ctrl + M` | Open Offline Metrics |
